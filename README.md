##html 
html (hipertext markup language) não é uma linguagem de programação, mas sim de marcação sevre para criar a estrutura de páginas web, sem interações complexas ou com decoração embelezamento. Serve para fazer tíyulos, parágrafos, listas, tabelas. Mas não é capaz por si só de  cor, o tamanho, direção, fonte etc. dos elementos

## css 
css(cascading style sheet) esse sim serve para fazer a parte  estética da página que o html não consegue. Ele ppode mudar a cor do fundo, a cor das letras, a disposição dos elementos, o tamanho da fonte, o sombreamento de elementos etc.
